## Data Dictionary

Please prepare a CSV file with this format:

| Variable | Content                 |
|----------|-------------------------|
| id       | unique identifier       |
| id2      | a non-unique identifier |
| name     | name of a cat           |
| age      | cat's age (if known)    |
